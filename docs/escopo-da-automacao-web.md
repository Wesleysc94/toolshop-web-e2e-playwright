# Escopo da Automacao Web

Esta etapa representa a primeira base de automacao web do portfolio.

O objetivo foi montar uma suite inicial que cobrisse os caminhos mais importantes da aplicacao sem transformar a primeira versao em algo pesado demais.

## Fluxos incluidos

- carregamento da home
- busca por produto
- abertura de detalhe de produto
- login
- logout
- tentativa de login com senha invalida
- adicao de item ao carrinho
- checkout ate a confirmacao de pagamento

## O que ficou de fora nesta primeira fase

- comparacao de produtos
- favoritos
- mensagens da conta
- fluxo completo de perfil
- variacoes de idioma
- execucao em multiplos navegadores
- regressao mais ampla

## Objetivo desta escolha

Entregar uma primeira suite web que seja:

- facil de entender
- simples de executar
- clara para revisar
- baseada em fluxos reais
- boa para evoluir nas proximas fases
